# ecommerce-sales-analysis
# 电商平台销售数据分析

## 项目简介
基于Kaggle公开数据集（Ecommerce_Sales_Data_2024_2025），使用Python完成端到端数据分析，涵盖数据清洗、探索性分析、时间序列预测、品类分析、地域分析与客户价值分层。

## 数据来源
- 来源：Kaggle
- 数据集：Ecommerce_Sales_Data_2024_2025.csv
- 许可证：CC0（公有领域）
- 记录数：5,000条订单
- 时间范围：2023-10 至 2025-10

## 分析方法
| 数据预处理 | 缺失值/重复值/异常值处理 |
| 销售趋势 | 时间序列分解、季节性指数 |
| 销售预测 | ETS、ARIMA(1,1,1) |
| 品类分析 | HHI指数、帕累托分析、折扣敏感度 |
| 地域分析 | ANOVA方差分析、箱线图、置信区间 |
| 用户分析 | 复购分析、客户价值分层 |

## 核心结论摘要
- ARIMA(1,1,1)预测未来半年月销售额稳定在2,250万元左右，MAE=182.5万
- 平台HHI指数=0.1003，品类高度分散，无单一品类依赖风险
- Furniture为“利润明星”品类，Books/Groceries适合引流
- 复购率仅3.2%，28.1%高价值客户贡献断层式领先
- 各区域销售额存在统计显著差异（p=0.026）但幅度不大，城市梯队分化明显
